Папка содержит код курсовой работы, посвященной использованию алгоритмов машинного обучения
для анализа финансовых новостей с целью спрогнозировать реакцию фондового рынка.

Оригинальные данные по новостям в формате json находятся в папке Original_data
(источник: Kaggle, https://www.kaggle.com/jeet2016/us-financial-news-articles)
Папка Stock_data содержит котировки рассматриваемых акций (источник: yahoo.finance)

Companies.py - полный список компаний, акции которых рассматриваются
Json_to_csv.py - создание единой базы новостей из json файлов (news_data.csv)
Data_preprocessing.py - создание единого датафрейма для последующего применения моделей машинного обучения

SVM_Bag-of-words.py, XGBoost_SenticNet.py и LSTM.py - представление текста в форме числовых векторов и применение 
моделей SVM, XGBoost и нейронной сети архитектуры LSTM
