Репозиторий содержит код моей курсовой работы, посвященной использованию алгоритмов машинного обучения для анализа финансовых новостей с целью спрогнозировать реакцию фондового рынка.

Оригинальная база данных состояла из 300 000+ новостей (каждая - в виде отдельного json файла) и была скачена с Kaggle (https://www.kaggle.com/jeet2016/us-financial-news-articles), котировки рассматриваемых акций скачены с yahoo.finance

Описание файлов:
Companies.py - полный список компаний, акции которых рассматриваются
Json_to_csv.py - создание единой базы новостей из json файлов (файл news_data.csv не был помещен в репозиторий из-за большого объема)
Data_preprocessing.py - создание единого датафрейма для последующего применения моделей машинного обучения (preprocessed.csv)
SVM_Bag-of-words.py, XGBoost_SenticNet.py и LSTM.py - представление текста в форме числовых векторов и применение моделей SVM, XGBoost и нейронной сети архитектуры LSTM
